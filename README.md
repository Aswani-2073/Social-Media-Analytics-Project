# Social Media Analytics for Strategic Branding

## Project Overview

This project focuses on analyzing social media data to understand user engagement, content performance, hashtag effectiveness, and campaign impact. The goal is to derive actionable insights that help improve brand visibility and engagement using data-driven decision-making.

---

## Tools & Technologies Used

* Microsoft Excel
* Pivot Tables
* Charts & Visualizations
* Data Cleaning Techniques

---

## Datasets Used

* Posts Data (likes, comments, shares, impressions, clicks)
* Campaign Metadata
* Weekly Engagement & Follower Data

---

## Project Tasks

### Task 1: Data Cleaning

* Removed duplicates
* Handled missing values
* Created cleaned datasets for analysis

### Task 2: Engagement Analysis

* Analyzed likes, comments, shares, and clicks
* Identified top-performing posts and platforms

### Task 3: Visualization

* Created bar charts, line charts, and pie charts
* Visualized engagement trends and content performance

### Task 4: Hashtag & Content Strategy

* Identified most-used hashtags
* Analyzed hashtag performance based on engagement
* Compared content types (image, video, carousel)

### Task 5: Campaign Effectiveness

* Measured engagement uplift before vs during campaigns
* Calculated ROI (Engagement vs Budget)
* Analyzed follower growth during campaigns

---

## Key Insights

* Video content showed higher engagement compared to other content types
* Certain hashtags consistently delivered higher reach and interactions
* Campaign periods resulted in noticeable engagement uplift

---

## Outcome

This project helped develop strong analytical thinking, Excel skills, and the ability to convert raw data into meaningful business insights. It reflects real-world social media analytics used in strategic branding.
